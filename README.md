# Barley’s Taproom & Pizzeria

Responsive one-page website for Barley’s Taproom & Pizzeria in Spindale, North Carolina.

The page prioritizes the food menu, rotating beer selection, phone ordering, live-music updates, hours, and location. Restaurant details and menu prices were checked against the current Barley’s website and Toast menu in July 2026; rotating taps and band dates intentionally direct visitors to the phone and Facebook because no authoritative live feed is currently available.

## Local development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run build
npm test
npm run lint
npx tsc --noEmit
```

The site uses the bundled vinext/Sites worker setup for local preview and hosting.
